## 方案二：程序过滤机制（补充方案）

### 新建文件
- `d:\AiPorject\AiStudyPartner\apps\web\src\services\format-filter.ts`

### 过滤规则设计

```typescript
// 需要过滤的符号模式
const SPECIAL_CHARS_PATTERNS = [
  /\*\*([^*]+)\*\*/g,           // **加粗** -> 普通文本
  /##\s*([^\n#]+)/g,            // ## 标题 -> 普通文本
  /#+\s*([^\n#]+)/g,            // # 各种标题 -> 普通文本
  /【([^】]+)】/g,              // 【重点】 -> 「重点」
  /\[([^\]]+)\]/g,              // [方括号] -> 「方括号」
  /\|([^|]+)\|/g,               // |竖线| -> 普通文本
  /`([^`]+)`/g,                 // `代码` -> 普通文本
  /```[\s\S]*?```/g,            // 代码块 -> 删除
  /<[^>]+>/g,                   // HTML标签 -> 删除
]

// 替换规则
const REPLACEMENTS = [
  { pattern: /【/g, replacement: '「' },
  { pattern: /】/g, replacement: '」' },
]
```
