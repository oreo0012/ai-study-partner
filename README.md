# AI学伴 - 儿童学习陪伴助手

一个基于 Vue 3 + TypeScript 的儿童学习陪伴应用。

## 项目结构

```
ai-study-partner/
├── apps/
│   └── web/                 # Web 应用
├── packages/
│   ├── stage-ui/            # UI 组件库
│   ├── stage-ui-live2d/     # Live2D 组件
│   ├── stage-shared/        # 共享工具
│   ├── i18n/                # 国际化
│   └── audio/               # 音频处理
└── package.json
```

## 快速开始

### 安装依赖

```bash
pnpm install
```

### 开发模式

```bash
pnpm dev
```

### 构建

```bash
pnpm build
```

## 技术栈

- Vue 3
- TypeScript
- Vite
- Pinia
- UnoCSS
