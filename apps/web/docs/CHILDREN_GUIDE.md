## 2. 查看我的任务

### 进入任务页面

1. 在首页点击"我的任务"入口
2. 进入任务页面

### 认识任务卡片

```
┌─────────────────────────────┐
│ ⭐ 练习          ○ 未完成  │  ← 类型和状态
│                             │
│ 完成习题: 数学练习题        │  ← 任务名称
│                             │
│ ⏱️ 30分钟                   │  ← 预计时间
└─────────────────────────────┘
```

### 任务状态说明

| 状态 | 图标 | 含义 |
|------|------|------|
| 未完成 | ○ | 还没做的任务 |
| 已完成 | ✓ | 已经完成的任务 |

### 进度展示

在任务页面顶部，可以看到今日学习进度：

```
┌─────────────────────────────┐
│ 📊 今日进度                 │
│                             │
│  2 / 5  (40%)             │  ← 完成情况
│ ████░░░░░░░░░░░░░░░░     │  ← 进度条
│                             │
│ ⏰ 45分钟    🔥 3天        │  ← 学习时长 | 连续天数
│                             │
│ ✨ 不错的开始！继续加油哦！  │  ← 鼓励语
└─────────────────────────────┘
```

### 任务分类

- **🎯 待完成**：还没做的任务，会显示在最前面
- **✨ 已完成**：已经做完的任务，会显示在后面

## 3. 完成任务

### 方法一：告诉AI学伴

1. 在首页和AI学伴聊天时
2. 告诉它你完成了某个任务，例如：
   - "我完成了数学练习"
   - "作业做完了"
   - "任务完成了"
3. AI学伴会问你确认
4. 确认后，任务状态会自动更新为"已完成"

### 方法二：点击任务卡片

1. 在任务页面点击要完成的任务
2. 如果是练习类任务，会进入习题练习模式
3. 完成练习后，任务自动标记为完成

> 💡 **提示**：告诉AI学伴是最简单的方式！你可以说"我今天完成了..."，AI会帮你记录。

### 题型说明

#### 选择题

1. 阅读题目和选项
2. 点击你认为是正确答案的选项
3. 系统会告诉你对错

**答题反馈**：
- 正确 🎉："回答正确！你真棒！"
- 错误 🤔：显示正确答案，并鼓励你"没关系哦，继续加油！"

#### 填空题

1. 阅读题目，注意空格位置
2. 在输入框中填写答案
3. 点击"提交答案"按钮

#### 简答题

1. 阅读题目
2. 在文本框中输入你的答案
3. 点击"提交答案"按钮

## 5. 和AI学伴聊天

### 基本对话

1. 在输入框中输入你想说的话
2. 点击发送按钮或按回车键
3. AI学伴会回复你

### 对话示例

**问好**：
- 你：你好！
- AI学伴：你好呀！今天我们要学习什么呢？

**请求帮助**：
- 你：我有题目不会
- AI学伴：没关系，告诉我是什么题目，我来帮你讲解！

**完成任务**：
- 你：我完成了数学作业
- AI学伴：你完成了数学作业吗？太棒了！你真是一个爱学习的好孩子！🌟

### AI学伴会记得你

AI学伴有很好的记忆力！它会：
- 记住你今天有哪些任务
- 记得你之前做错的题目
- 在适当的时候提醒你完成任务
- 给你鼓励和建议

### 鼓励语示例

当你完成任务时，AI学伴会说：

- "太棒了！你完成了今天的任务，继续加油呀！🌟"
- "太厉害了！你真是一个爱学习的好孩子！⭐"
- "太棒了！你已经完成了一个任务哦，继续保持！✨"
- "哇！你好棒呀！完成任务的你超级厉害！🎉"
